# my-first-vue-project

> 这是一个vue的练习
> 做了一个页面，实现自动添加list的功能

#用到的技术点

	1. v-for 做一个list 集合
	2. v-bind 指令  对class的控制显示
	3. v-on:click 点击事件（状态互换）
	4. v-model 双向绑定，输入的值放入节点里面
	5. 监听 watch
	6. localStrore 浏览器的本地存储
	7. 父子组件的传递参数，组件之间的通信
